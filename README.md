# Monster Hunter Rise Elements Mod

This project is a Lua mod for Monster Hunter Rise using REFramework.

## Features

- modifies elements/status behavior
- increases efficacy
- modifies visuals of elements/status
-![Example](screenshots/2026_01_17_230720.png)


## Technologies

- Lua
- REFramework

## Installation

1. Install REFramework
2. Copy the script into the autorun folder
